# PhoneInput

Input field for mobile number with selected country option value 

### Usage

```js
import { PhoneInput } from "app-components";
```

```jsx
<PhoneInput
  locale={locale}
/>
```

### Properties

| Props               |   Type   | Required |             Values              | Default | Description                                                                                                                                                                                                                                                   |
| ------------------- | :------: | :------: | :-----------------------------: | :-----: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `locale`             | `string` |    ✅    |                -                |    -    | Locale value equal ISO-2 code standart                                                                                                                                                                                                                                            |
